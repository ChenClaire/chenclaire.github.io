  My Experience

(4) Collaborated closely with the back-end developer to implement data into front-end and improve code quality.

  User Interview & Problem Identification

We had interviews with 5 agents and came back with tons of insights about their current workflow and pain points. We identified existing problems from the notes, and generated a user journey map.

Problems #1: Searchability is weak: current model of document retrieval is cumbersome, content is hard to find.

Problems #2: An easy access to top or frequently used resources is lacking.

Problems #3: Mobile unfriendly: hard to use on the go.

  Solutions

#1 Powerful Search. When searching by keywords, an agent can refine his search by sections or categories, which means he can search within or across sections.

#2 Featured Links. Links (or documents) in each section are ordered by views and date.

#3 Responsive Design. An agent can access all features on mobile and desktop.
